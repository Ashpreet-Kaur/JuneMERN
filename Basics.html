HTML Basics

HTML stands for HyperText Markup Language. It is made of tags that wrap text or other content.

Common HTML tags

Most tags come in pairs: an opening tag and a closing tag. For example, the opening heading tag is written as h2 and the closing heading tag is written as /h2. The opening tag starts the element, and the closing tag ends it.

Examples of basic tags

  • Heading tags: from h1 to h6.
  • Paragraph tag: p.
  • Button tag: button.

Some tags do not need a closing tag, such as
or . These tags are used alone because they insert content directly.
