CSS Typography Properties

Typography in CSS is used to control the appearance of text on a web page. We can change the font, size, style, alignment, spacing and many other text-related properties to improve readability and design.

Important Typography Properties

  • font-family
  • font-size
  • font-weight
  • font-style
  • text-align
  • text-transform
  • text-decoration
  • letter-spacing
  • word-spacing
  • line-height
